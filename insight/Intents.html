Intents

Intent:
  • An Intent is a messaging object you can use to request an action from another app component.
  • Intents facilitate communication between components in Android App.
  • To start an activity, intent is passed to startActivity() or startActivityForResult().
  • To start a service, intent is passed to startService() or bindService(). Intent describes the service to start and carries any necessary data.
  • To deliver a broadcast, intent is passed to sendBroadcast(), sendOrderedBroadcast(), or sendStickyBroadcast().
Intent Types:

There are two types:
  • Explicit intents specify the component to start by name.

    Example:
    String imgUrl = "http://www.test.com/sample_image.png"
    Intent intent = new Intent(ActivityName.this, DownloadService.class);
    intent.setData(Uri.parse(imgUrl));
    startService(intent);

  • Implicit intents do not name a specific component, but instead declare a general action to perform, which allows a component from another app to handle it.

    Example:
    String appPackage = "com.example.demoApplication";//your application package name
    Intent intent = new Intent(Intent.ACTION_VIEW, Uri.parse("market://details?id=" + appPackage);
    startActivity(intent);
